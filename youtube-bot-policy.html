Политика конфиденциальности

Я не сохраняю и не передаю никакие данные, полученные через API YouTube. Все запросы выполняются в момент обращения, данные не хранятся и не обрабатываются третьими лицами.

1. Какие данные я собираю

Я сохраняю предоставленные вами API-ключи YouTube (ключи доступа к YouTube Data API), чтобы выполнять запросы от вашего имени.

2. Как я использую эти данные

API-ключи используются только для выполнения запросов к YouTube API от вашего имени. Также они могут использоваться для подсчёта количества запросов и соблюдения ограничений.

3. Как я храню и защищаю данные

Все ключи хранятся в базе данных с ограниченным доступом и защищены средствами шифрования/хэширования. Я не передаю эти ключи третьим лицам.

4. Удаление данных

Вы можете запросить удаление вашего API-ключа, написав мне в Telegram.

5. Сторонние сервисы

Бот использует только официальное API YouTube. Ваши ключи не передаются в сторонние системы, кроме YouTube Data API.

6. Изменения в политике

Я могу обновлять политику конфиденциальности. Актуальная версия всегда доступна по этой ссылке.
