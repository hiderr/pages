Политика конфиденциальности

Бот не сохраняет и не передает никакие данные, полученные пользователем через API YouTube. Все запросы выполняются в момент обращения, данные не хранятся и не обрабатываются третьими лицами.

1. Какие данные собирает бот

Бот сохраняет предоставленные пользователем API-ключи YouTube (ключи доступа к YouTube Data API) в базе данных, чтобы выполнять запросы от имени пользователя.

2. Как бот использует эти данные

API-ключи используются только для выполнения запросов к YouTube API от имени пользователя.

3. Как бот хранит и защищает данные

Все ключи хранятся в базе данных с ограниченным доступом и защищены средствами шифрования. Ключи не передаются третьим лицам.

4. Удаление данных

Пользователь может запросить удаление API-ключа, написав разработчику в Telegram.

5. Сторонние сервисы

Бот использует только официальное API YouTube. Пользовательские ключи не передаются в сторонние системы, кроме YouTube Data API.

6. Изменения в политике

Разработчик оставляет за собой право вносить изменения в настоящую политику конфиденциальности без предварительного уведомления пользователей. Актуальная версия политики всегда доступна по этой ссылке. Продолжение использования бота после внесения изменений означает согласие пользователя с обновлённой политикой.

7. Отказ от ответственности

Сервис предоставляется "как есть" без каких-либо явных или подразумеваемых гарантий. Разработчик не несёт ответственности за любые прямые или косвенные убытки, возникшие в результате использования бота, включая, но не ограничиваясь, сбоями в работе API YouTube, ограничениями со стороны сторонних платформ, а также действиями пользователей.

Пользователь самостоятельно несёт ответственность за соблюдение всех применимых законов, включая, но не ограничиваясь, авторским правом, условиями платформ, а также политиками API YouTube.
