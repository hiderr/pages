Политика конфиденциальности

Бот не сохраняет и не передает никакие данные, полученные пользователем через API YouTube. Все запросы выполняются в момент обращения, данные не хранятся и не обрабатываются третьими лицами.

1. Какие данные собирает бот

Бот сохраняет предоставленные вами API-ключи YouTube (ключи доступа к YouTube Data API) в базе данных, чтобы выполнять запросы от вашего имени.

2. Как бот использует эти данные

API-ключи используются только для выполнения запросов к YouTube API от вашего имени.

3. Как бот хранит и защищает данные

Все ключи хранятся в базе данных с ограниченным доступом и защищены средствами шифрования. Ключи не передаются третьим лицам.

4. Удаление данных

Вы можете запросить удаление вашего API-ключа, написав разработчику в Telegram.

5. Сторонние сервисы

Бот использует только официальное API YouTube. Ваши ключи не передаются в сторонние системы, кроме YouTube Data API.

6. Изменения в политике

Я, разработчик бота, могу обновлять политику конфиденциальности. Актуальная версия всегда доступна по этой ссылке.

7. Отказ от ответственности

Бот предоставляется "как есть". Я, разработчик, не гарантирую бесперебойную или безошибочную работу сервиса, а также не несу ответственности за возможные сбои в работе YouTube API, ограничения со стороны платформы, или некорректные действия, совершённые пользователями с использованием API-ключей.
